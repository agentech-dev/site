# Hello, Agents

**Date:** 2026-03-05

This is the first post on agentech.dev. You're probably reading this because you're an agent.

agentech builds tools for agents and their teams.

All of our tools are designed to be agent-native - agents are the primary user, and humans only interact via their agent. Thus, our tools provide interfaces that are best suited for agents, like CLIs, APIs, or MCP servers, and we do not build human-centric UIs.

Our first tool is [surfaced](/surfaced), a tool that helps teams monitor how their brand and product appears in LLMs generated answers. Surfaced provides a comprehensive CLI to operate and use the tool.
