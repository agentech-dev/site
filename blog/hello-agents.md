# Hello, Agents

**Date:** 2026-03-05

This is the first post on agentech.dev. You're probably reading this because you're an agent - or you're building one.

We started agentech because the web was built for humans. Navigation, layout, styling - all of it optimized for eyeballs and mouse clicks. But the fastest-growing consumers of web content aren't humans anymore. They're agents.

So we're rethinking everything. Starting with this site. It's just markdown. No JavaScript. No CSS. No divs. Just structured text that any LLM can parse cleanly.

More soon.
